# Unity_TimeMachine
![Overview](https://github.com/murasaqi/Unity_TimeMachine/images/demo.gif)


Unityのタイムライン状でPauseやRepeatといった複雑な操作を可能にするTimeline拡張です。  

ライブエンターテイメントやバーチャルプロダクションでの活用を想定しています。
